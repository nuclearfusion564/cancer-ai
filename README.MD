# Cancer-AI
This language model can detect if someone has brain cancer with certain data. The dataset used is here: https://www.kaggle.com/datasets/jakeshbohaju/brain-tumor

# DISCLAMER
DO NOT RELY ON THIS MODEL, THIS WAS MADE FOR EDUCATIONAL PURPOSES ONLY. THIS ISN'T PROFESSIONALLY MADE.

To run and play around with on your computer just run

```bash
git clone https://github.com/nuclearfusion564/cancer-ai.git
cd cancer-ai 
npm install
```
Then you can easily edit things in your IDE and to test just run:
```bash
npm test
```
